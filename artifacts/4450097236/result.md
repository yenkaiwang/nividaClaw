## Qwen 3.6 是什麼

Qwen（通義千問）是由阿里巴巴集團旗下通義實驗室開發的大型語言模型系列。Qwen 3.6 是其中的最新版本，參數規模為 350 億（35B），屬於中等規模的高效模型。

## 此 Model 的主要功能

1. **程式碼生成與理解**：支援多種程式語言，能根據自然語言描述生成程式碼，也能閱讀、解釋、除錯既有程式碼。

2. **自然語言對話**：能夠回答提問、進行對話、總結文件或翻譯文字（支援繁體中文、簡體中文、英文等多種語言）。

3. **程式碼除錯與優化**：分析程式碼問題、提出修正建議、優化效能或可讀性。

4. **技術文件撰寫**：協助撰寫程式註解、技術文件、API 文件等。

## 在此環境中的運作方式

此模型透過 **pgx 网关**（https://pgx.playwright.tw/v1）提供服務，以 OpenAI 相容的 API 格式進行請求與回應。

## 適用場景

- 日常程式開發輔助
- 技術問題解答
- 程式碼審閱與建議
- 文件撰寫與翻譯
- 創意發想與討論
