# Model 說明：Qwen 3.6 (35B)

## 基本資訊

- **模型名稱**：Qwen 3.6
- **參數量**：35B（約 350 億參數）
- **開發者**：阿里巴巴集團旗下的通義實驗室（Tongyi Lab）
- **部署位置**：保哥的 Lenovo ThinkStation PGX 伺服器
- **服務端點**：https://pgx.playwright.tw/v1

## 主要功用

### 1. 自然語言處理
Qwen 3.6 是一個大型語言模型（LLM），能夠理解並生成自然語言文字，支援中文、英文等多種語言。

### 2. 程式碼生成與理解
- 支援多種程式語言的程式碼生成
- 程式碼理解、審查、與除錯
- 適合協助開發者進行程式開發工作

### 3. 邏輯推理
- 具備強大的邏輯推理能力
- 在數學計算、科學推理等領域表現優異

### 4. 對話與問答
- 能夠進行多輪對話
- 回答各類知識性問題
- 提供清晰、易懂的解释

## 在本專案中的用途

在本專案（nividaClaw / GitHubClaw）中，Qwen 3.6 主要作為 **GitHubClaw 工具包的 AI 引擎**，用來說明、處理與自動化 GitHub Issue 相關的任務。
