# 🛠️ Felo Slides（AI 簡報產生器）

> 只要一句話，小龍蝦就幫你生出一整份簡報，還附上可分享的連結！

## ✨ 這個技能可以做什麼？

- 輸入主題或大綱，自動產生一份完整的簡報投影片
- 把你的筆記、報告重點快速轉成簡報 deck
- 產生完成後回傳一個可直接分享的投影片網址
- 支援用自然語言描述你想要的簡報內容與風格

## 📦 安裝方式

在 Telegram 對話中輸入：

```
/skills
```

輸入後會顯示可安裝的技能列表，選擇此技能即可完成安裝。

## ⚙️ 需要的設定

| 設定項目 | 說明 |
| --- | --- |
| `FELO_API_KEY` | Felo 平台的 API 金鑰，可到 [felo.ai](https://felo.ai) 申請 |

## 💬 提示詞範例

```text
幫我做一份關於「人工智慧在教育領域的應用」的簡報，大約 8 頁
把以下筆記整理成一份簡報：1. 專案背景 2. 技術架構 3. 成果展示 4. 未來規劃
幫我生一份 5 頁的自我介紹簡報，包含學歷、專長和作品集
做一份關於全球暖化的英文簡報，適合高中生報告用
```

## 📝 輸出範例

小龍蝦會回傳一個可以直接打開的簡報連結，像這樣：

```
✅ 簡報產生完成！
🔗 https://ppt.felo.ai/share/abc123xyz
```

點開連結就能在瀏覽器中查看、下載或分享你的簡報。

## ⚠️ 注意事項

- 簡報產生需要一點時間（通常 1～3 分鐘），請耐心等候
- 產生的簡報會託管在 Felo 平台上，需要網路才能查看
- 內容品質取決於你的提示詞——描述越詳細，簡報越精準
- 需要先設定好 `FELO_API_KEY` 才能使用
