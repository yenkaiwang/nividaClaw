# 🛠️ Find Skills（技能探索）

> 不知道該用哪個技能？讓小龍蝦幫你找到最適合的那一個！

## ✨ 這個技能可以做什麼？

- 根據你的需求，推薦最適合的技能（skill）
- 分析你的任務目標，找出需要哪些能力
- 搜尋所有可用的技能，幫你篩選出最佳選項
- 當任務需要多個技能搭配時，建議使用順序
- 提供推薦理由和設定說明，讓你馬上就能開始用

## 📦 安裝方式

在 Telegram 對話中輸入：

```
/skills
```

輸入後會顯示可安裝的技能列表，選擇此技能即可完成安裝。

## 💬 提示詞範例

```text
我想要把網頁內容抓下來，應該用哪個技能？
有沒有可以幫我做簡報的技能？
我需要搜尋 Twitter 上的資訊，推薦我用什麼
我的任務是先蒐集資料再做成簡報，應該怎麼搭配技能？
有哪些技能可以用？幫我列一份清單
```

## 📝 輸出範例

小龍蝦會根據你的需求給出推薦，像這樣：

```
🎯 根據你的需求，推薦使用以下技能：

1. felo-web-fetch — 擷取網頁內容，適合你要抓取文章資料的需求
2. felo-slides — 把整理好的資料轉成簡報

💡 建議順序：先用 felo-web-fetch 蒐集資料，再用 felo-slides 產生簡報。
⚙️ 這兩個技能都需要設定 FELO_API_KEY。
```

## ⚠️ 注意事項

- 這是一個「找技能的技能」（meta-skill），本身不會執行其他任務
- 不需要任何 API 金鑰，直接就能使用
- 描述需求越具體，推薦結果越精準
